# Linguistic Help 🇮🇹🇬🇧

A simple web app that helps you communicate in Italian and English in real time.

## ✨ Features

* 🎤 Speak in Italian → get English translation
* 💬 AI-generated reply suggestions
* ✍️ Type replies in English → translate to Italian
* 🔊 Text-to-speech (Italian voice output)

## 🚀 Use Case

Designed for:

* Job seekers in Italy
* University students
* Everyday conversations

## 🛠️ Tech Stack

* Frontend: HTML, Tailwind CSS, JavaScript
* Backend: Supabase Edge Functions
* Translation: LibreTranslate
* AI: OpenRouter (Mistral model)

## ⚙️ Setup

1. Clone the repo:

```
git clone https://github.com/yourusername/linguistic-help.git
```

2. Open `index.html`

3. Replace:

```
YOUR_SUPABASE_FUNCTION_URL
```

with your deployed Supabase function URL.
